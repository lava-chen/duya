**v0.4.1-beta.1** (2026-07-10) — first beta on the `0.4.1` line.

Headline of this beta is **macOS as a first-class release target**.
The release workflow now builds, signs, and notarizes a real `.app`
bundle via a two-pass CI build, and the agent/electron layers have
been hardened for darwin (shell detection, node probe, app menu,
tray, proxy). A new in-app browser panel with a CDP-backed webview
daemon rounds out the feature set.

**Highlights**

- **macOS packaging, signing, and two-pass build**
  - GitHub Actions release workflow extended with a `macos-latest`
    matrix leg producing a signed + notarized `.app` / `.dmg`.
  - Two-pass `build:agent` breaks the `@duya/agent ↔
    @duya/conductor` circular dependency on a clean checkout.
  - macOS code signing + notarization via `after-sign.js` when
    `CSC_LINK` / `APPLE_*` secrets are present (ad-hoc unsigned
    fallback otherwise).

- **macOS desktop shell (`feat(electron)`)**
  - Native macOS app menu, tray icon, system proxy detection, and
    node runtime detection.
  - `fix(agent): harden write tool and shell detector for macOS`
    makes path / shell handling correct on darwin.
  - `fix(cli): probe dev and prod userData dirs for runtime config`
    so the standalone CLI resolves config on macOS.

- **In-app browser panel (`feat(browser)`)**
  - Webview CDP client + daemon endpoint for agent-driven browsing.
  - Agent-driven webview tab in the sidebar panel.
  - Cookie importer with DPAPI decryption (Windows).
  - Backend degradation chain + settings UI, with an inline backend
    toggle in the browser panel toolbar.
  - `fix(browser): live-update backend mode on running agents
    without re-init`.

- **Agent core**
  - MCP security layer with env allowlist and secret redaction.
  - Gateway relay and cron agent profiles.
  - Canvas tool sizing guidance overhaul for compact tiers.
  - `fix(agent): harden MiniMax provider and prevent base64 prompt
    leaking`.

- **Gateway**
  - Voice / file / video media support end-to-end.
  - `fix(gateway): resolve workspace config and harden session
    lifecycle`.

- **Conductor / canvas**
  - Remove passive collision and merge widgets into the freeform
    layer.

- **UI**
  - File preview screenshot and sidebar folder open.
  - `fix(ui): prioritize session canvas binding and expand sub-agent
    tool view`.
  - Panel sizing, text truncation, and widget css refinements.

- **Docs**
  - README rewrite and architecture / exec-plans refresh.

**Breaking changes**

None. All changes are additive; existing IPC / CLI / agent surfaces
are unchanged.

**Pre-release verification**

- `npm run typecheck:all` — 0 errors.
- `npm run electron:build` — exit 0, bundle artifacts present.
- macOS packaged build smoke test pending (reason for `-beta.1`).
