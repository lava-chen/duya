# Literature Plugin

Provides literature asset management, evidence tracking, and citation tools for research workflows.

## What this plugin adds

- `literature:add_source` tool — add literature sources (papers, books, articles)
- `literature:search_sources` tool — search through collected sources
- `literature:search_evidence` tool — search evidence spans across sources
- `literature:get_citation` tool — format citations (bibtex, apa, gbt7714)
- `literature:extract_paper_card` tool — extract structured paper metadata

## When to suggest this plugin

Suggest when the user is doing academic research, literature review, paper writing, or needs to manage references and citations.
