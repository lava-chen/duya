# Citation Format

Format citations in various academic styles using the literature plugin.

## Supported Styles

- **bibtex** — BibTeX format for LaTeX documents
- **apa** — APA 7th edition style
- **gbt7714** — GB/T 7714 Chinese national standard

## Usage

Use the `literature:get_citation` tool with the source ID and desired style.

## When to Use

- When preparing references for a paper
- When converting between citation formats
- When building a bibliography for a research document
