# Paper Analysis

Analyze academic papers with structured reading and evidence extraction.

## Process

1. Read the paper abstract and introduction to understand the core problem
2. Identify the methodology and experimental setup
3. Extract key findings and contributions
4. Note limitations and future work mentioned by authors
5. Cross-reference citations with other sources in the literature database

## Tools Used

- `literature:search_sources` — find related papers
- `literature:search_evidence` — find supporting or contradicting evidence
- `literature:extract_paper_card` — store structured metadata

## Output Format

Produce a structured analysis with:
- Problem statement
- Methodology summary
- Key findings
- Limitations
- Relevance to current research
